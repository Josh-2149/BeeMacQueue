## 🔔 Complete Notification Flow

| Action | Who Gets Notified |
|--------|------------------|
| Customer joins queue | ✅ Customer (self) + ✅ All Staff |
| Customer leaves queue | ✅ Customer (self) + ✅ All Staff |
| Staff creates queue | ✅ All Customers in branch + ✅ All Staff |
| Staff serves next | ✅ The specific Customer being served |
| Staff marks served | ✅ The specific Customer |
| Staff cancels customer | ✅ The specific Customer |
| Staff calls customer | ✅ The specific Customer |

## 🎨 Customer Queue Card States

| State | Button Text | Color | Condition |
|-------|------------|-------|-----------|
| Available | **Join Queue** ➜ | Red | Queue not full, user not in any queue |
| Your Queue | ✓ **Your Queue** | Green | User is actively in this queue |
| Served Today | ✓ **Served Today** | Blue | User completed this queue today |
| **Queue Full** | **Queue Full** ⚠ | Orange | waiting + serving >= capacity |
| Joining | **Joining...** ⌛ | Gray | Currently processing join request |
| In Another Queue | **In Another Queue** | Gray | User has active queue elsewhere |
